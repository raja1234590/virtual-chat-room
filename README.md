💬 Live-Talk-AI – Real-Time Chat Application

Live-Talk-AI is a full-stack real-time chat application built using the MERN stack with Socket.IO for instant messaging.
It supports one-to-one chats, group chats, file sharing, and real-time notifications with a clean, responsive UI.

🚀 Features
🔐 Authentication

Secure JWT-based authentication

User registration & login

Protected routes

💬 Chat System

One-to-one private chats

Group chats with admin controls

Create, rename, add/remove users in groups

Latest message preview

⚡ Real-Time Messaging

Instant message delivery using Socket.IO

Real-time notifications

Live chat updates without refresh

📎 File Sharing

Upload and send files (PDF, images, docs)

Clickable file previews

👤 User Profiles

View user profile (name, email, avatar)

Profile modal for chat participants

🧠 Smart UI

Responsive design using Chakra UI

Message alignment & sender grouping

Smooth scrolling chat feed
